# README

## GUI

Ensure the GUI window is in focus before using the keys.

- **W**: Move forward.
- **S**: Move backward.
- **A**: Move left.
- **D**: Move right.

- **Up Arrow**: Tilt up.
- **Down Arrow**: Tilt down.
- **Left Arrow**: Pan left.
- **Right Arrow**: Pan right.
